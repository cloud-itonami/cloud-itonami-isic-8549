# Operator Guide

## First Deployment

1. Register the operator's license, jurisdiction and responsible educators/staff.
2. Import historical student/participant records and counterparties.
3. Run read-only validation of existing records against this blueprint's
   contracts.
4. Configure the Instruction Integrity Governor's hold/escalation policy.
5. Publish a dry-run operation and audit export.

## Minimum Production Controls

- licensed-educator sign-off required before any determination
- finalizing a certification or completion record always requires a human sign-off
- audit export for every hold, approval and record action
- backup manual process for governor/system outage

## Certification

Certified operators must prove student/academic-record integrity, governor
independence, evidence-backed reporting and human review for every
high-stakes action.
